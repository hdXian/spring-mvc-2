1. 표준 HTML 주석

html 주석은 html 주석임. 뷰 템플릿과 연관 없음. 렌더링 시에도 그대로 남고, 브라우저 페이지 소스 보기에도 남아있음.

2. 타임리프 파서 주석

기본적으로 html 주석으로 감싸져 있으나, 렌더링 과정에서 뷰 템플릿에 의해 주석 부분이 지워짐.
html 주석으로 감싸져 있으므로 html 파일을 그대로 열면 해당 부분이 보이고,
렌더링된 페이지를 브라우저에서 볼 경우 해당 부분이 없음. (페이지 소스를 통해 볼 수 없음.) 우선 html 주석으로 감싼 뒤, 다시 안의 내용을 /* */ 로 감싸서 표현.

3. 타임리프 프로토타입 주석

렌더링 시 해당 부분을 정상 출력하게 만들고,
html 파일로 직접 열 경우 주석처리.
즉 렌더링을 거쳤을 경우에만 보이도록 하는 기능.

어렵게 생각하지 말고, html 파일로 직접 열 경우 그냥 다 주석처리 돼서 그대로 보임. 기본적으로 html 주석으로 감싸놓은 형태고, 렌더링을 거치지 않기 때문.
차이는 렌더링을 거쳤을 경우에 나타남. 2, 3번 경우는 렌더링 시 해당 주석 부분을 아예 소스에서 지워버리거나(2), 오히려 화면에 표시하도록(3) 한다는 차이가 있음.
